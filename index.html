Team Project

MovieKnight

Description

This application is a fullstack application a small team and I made. The concept is an application where you can make multiple groups to watch movies with. These groups can add movies to the group watchlist and then set an event to watch the movie. We used a free movie API to search and add movies to any list of the users choosing. We used a document database to store our data (MongoDb). My role was product owner. Many of the decisions involving the direction of our application were left to me. As far as my responsibility for the code, I handled most of the frontend UI logic and some of the backend routes that were missed in our earlier sprints. Our team practice Scrum/Agile during the development of MovieKnight
